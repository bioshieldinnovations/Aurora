AURORA SUITE · v2.0

Tool suite with an aurora borealis web interface.

QR Generator · Cryptographic Encryption · Universal Downloader.
Installation
Bash

# 1. Clone / extract the project
cd aurora/

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run
python app.py

The browser opens automatically at http://localhost:5000.
Tools
⬡ QR Generator

    QR codes from URLs, plain text, or any string

    Customizable colors (fill + background)

    Rounded modules with adaptive error correction

    Direct download in PNG format

◈ Message Encrypter

    Symmetric stream encryption using PBKDF2-HMAC-SHA256

    Integrity verification with HMAC before decryption

    100,000 key-stretching iterations

    Random 16-byte salt per operation

    Clean hexadecimal output

↓ Universal Downloader

    Compatible with YouTube, Twitter/X, Reddit, Vimeo, TikTok, and 1000+ sites

    Preview with thumbnail, title, channel, and duration before downloading

    Video Mode (maximum quality MP4) or Audio Mode (192kbps MP3)

    Real-time progress bar

Cryptographic Security

The encryption module uses an Encrypt-then-MAC scheme:

    A key is derived using PBKDF2-HMAC-SHA256 (100k iterations + random salt)

    Encrypted using a stream XOR generated by SHA-256 in counter mode

    An HMAC-SHA256 of the ciphertext is attached to verify integrity

    The final package: SALT(16B) + HMAC(32B) + ENCRYPTED_DATA

GUARANTEED: an incorrect password is detected before attempting decryption.
Important Aspects

    Downloads are saved to ~/Aurora_Downloads/

    Encrypted files use the .aur extension

    The server listens on 0.0.0.0:5000 (accessible on the local network)
